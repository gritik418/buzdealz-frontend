# Buzdealz Frontend

Buzdealz is a premium e-commerce deal aggregator designed to help users find the best deals across various platforms with real-time price tracking and alerts.

**ðŸ”— [Live Demo](https://buzdealz-frontend.vercel.app/)**

## Tech Stack
- **Framework**: React 18 with Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **State Management**: Redux Toolkit & React Query
- **Animations**: Framer Motion & GSAP
- **Routing**: Wouter


## Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Configure Environment**:
   Create a `.env` file with:
   ```env
   VITE_API_URL=your_backend_api_url
   ```

3. **Run Development Server**:
   ```bash
   npm run dev
   ```

4. **Build for Production**:
   ```bash
   npm run build
   ```
